

# Mbote Foundation Website

## Overview
A multi-page, modern, credible website for Mbote Foundation — an Africa-focused innovation and venture organization. Design inspired by YC's clean, minimal aesthetic with the brand's African green and gold palette.

## Brand & Design
- **Colors**: Deep African Green (#0B6E4F), Warm Gold (#F5B700), Off-White (#FAFAFA), Charcoal (#111111)
- **Typography**: Inter font throughout, bold headlines, clean body text
- **AI-generated logo**: Modern mark + "Mbote Foundation" wordmark
- **AI-generated hero image**: African innovation/tech theme
- **Style**: Minimal, lots of whitespace, smooth scroll animations, card-based layouts

## Pages & Structure

### Shared Layout
- **Navbar**: Logo, nav links (About, Focus Areas, Ventures, Builders, Contact), mobile hamburger menu
- **Footer**: Locations, social link placeholders, tagline

### 1. Home Page (`/`)
- **Hero**: Full-width section with AI-generated background, headline "Building the Future from Africa.", subheadline, two CTA buttons (Apply to Build, Join the Community)
- **Mission snippet**: Brief about section with link to full About page
- **Focus Areas preview**: Grid of 8 cards (Education, Health, Climate, Energy, Entrepreneurship, Agriculture, Emergency Aid, Sports/Arts/Cinema) with icons
- **Future Vision timeline**: 2025→2040 visual timeline
- **CTA banner**: "Get in touch" section

### 2. About Page (`/about`)
- Mission statement with full description
- Headquarters & innovation hub locations with a visual layout
- Organization values/philosophy

### 3. Ventures Page (`/ventures`)
- Mbote Ventures description
- Grid of 12 investment areas (AI, Robotics, Fintech, Blockchain, Renewable Energy, Biotech, Healthcare, Space Tech, Manufacturing, Climate Tech, AgriTech, Infrastructure Tech) with icons and descriptions
- Bold CTA: "Founders building the future should reach out."

### 4. Builders Page (`/builders`)
- Mbote Builders community description
- Features: hackathons, research labs, startup incubation
- Two CTAs: Join Builders, Partner With Us

## Technical Details
- React + Vite + Tailwind CSS + TypeScript
- React Router for multi-page navigation
- Lucide icons for focus area and venture cards
- Subtle CSS animations (fade-in on scroll using Intersection Observer)
- Fully responsive mobile-first design
- AI-generated hero image and logo via Lovable AI image generation, saved as static assets
